Ukuran Kanal C<\/h2>\n

Kanal C mempunyai ukuran panjang 6 meter. Ukuran ini ialah ukuran yang beredar di pasaran dengan ketebalan hingga 1,0 mm. Penggunaan Kanal C bisa disesuaikan dengan atap yang akan dipakai. Jika atapnya ringan, maka Kanal C tak perlu terlalu kuat (tebal), sehingga mampu menghemat biaya anggaran pembangunan.<\/p>\n

Kegunaan Kanal C<\/h2>\n

Pemakaian Kanal C yang paling utama adalah sebagai rangka atap \/ dudukan atap secara langsung. Biasanya konstruksi gedung-gedung besar yang menggunakan rangka atap berupa Kanal C.<\/p>\n

Namun, untuk konstruksi perumahan biasa menggunakan balok kayu sebagai rangka penahan genteng. Namun, tidak dapat dipungkiri kemungkinan penerapan Kanal C pada konstruksi atap perumahan masa sekarang, mengingat Kanal C ini sungguh-sungguh praktis juga didesain dengan ketahanan yang sungguh-sungguh kuat. Seringkali rangka atap Kanal C kita temukan pada rumah modern ketika membikin kanopi ataupun gazebo.<\/p>\n
