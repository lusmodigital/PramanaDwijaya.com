Ukuran Kanal C<\/h2>\n

Kanal C yang beredar dipasaran umumnya memiliki panjang 6 meter dan mencapai 1,0mm ketebalannya. Pemakaian Kanal C dapat disesuaikan dengan atap yang akan diterapkan. Apabila atapnya ringan, Kanal C tidak perlu terlalu kuat (tebal), sehingga sanggup menghemat dana anggaran pembangunan.<\/p>\n

Kegunaan Kanal C<\/h2>\n

Kanal C digunakan terutama untuk pembuatan rangka atap yang akan bersentuhan langsung dengan atap. Pemilihan Kanal C sebagai rangka atap lazimnya dipakai pada konstruksi gedung-gedung besar, pabrik, dan lain-lain.<\/p>\n

Namun, Kanal C lazimnya digantikan dengan balok kayu pada konstruksi rumah-rumah hunian. Akan tetapi, tak menyangkal kemungkinan penggunaan Kanal C pada konstruksi atap perumahan masa kini, mengingat Kanal C ini sangat praktis juga didesain dengan ketahanan yang sangat kuat. Kanal C pada konstruksi perumahan biasa terdapat pada rangka atap Gazebo maupun Kanopi.<\/p>\n
