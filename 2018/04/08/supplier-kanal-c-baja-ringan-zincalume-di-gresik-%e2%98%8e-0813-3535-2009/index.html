Supplier Kanal C Baja Ringan Zincalume di Gresik ☎ 0813-3535-2009

Kanal C memiliki penampakan seperti balok panjang dengan bentuk muka seperti huruf C dan terbuat dari bahan baja ringan. Kanal C memiliki nama lain CNP, Balok Purlin, C Chanel dan Profil C. Kanal C dibuat dengan proses fabrikasi melalui tahap cutting dan bending dengan bahan plat koil.

Ukuran Kanal C

Kanal C mempunyai ukuran panjang 6 meter. Ukuran ini ialah ukuran yang beredar di pasaran dengan ketebalan hingga 1,0 mm. Penggunaan Kanal C bisa disesuaikan dengan atap yang akan dipakai. Jika atapnya ringan, maka Kanal C tak perlu terlalu kuat (tebal), sehingga mampu menghemat biaya anggaran pembangunan.

Informasi lebih lengkap perihal ukuran dan varian Kanal C bisa Anda lihat disini.

Kegunaan Kanal C

Penerapan Kanal C yang paling utama yaitu sebagai rangka atap / dudukan atap secara langsung. Pemilihan Kanal C sebagai rangka atap lazimnya dipakai pada konstruksi gedung-gedung besar, pabrik, dan lain-lain.

Namun, Kanal C lazimnya digantikan dengan balok kayu pada konstruksi rumah-rumah hunian. Namun, tidak dapat dipungkiri kemungkinan penerapan Kanal C pada konstruksi atap perumahan masa sekarang, mengingat Kanal C ini sungguh-sungguh praktis juga didesain dengan ketahanan yang sungguh-sungguh kuat. Seringkali rangka atap Kanal C kita temukan pada rumah modern ketika membikin kanopi ataupun gazebo.
